# Orbit Bastion

A lightweight browser tower defense game built from scratch.

## What it does

- Defend a winding route with three tower types.
- Earn credits from each cleared wave and invest in stronger defenses.
- Progress across planets: Mercury, Venus, Mars, Europa, and Neptune.
- Each new planet increases enemy durability, speed, wave count, and rewards.

## How to run

1. Open [index.html](./index.html) in a browser.
2. Select a tower type from the left panel.
3. Click glowing pads on the battlefield to build towers.
4. Press `Start Wave` to launch enemies.
5. After clearing all waves on a planet, use `Travel To Next Planet`.

## Files

- `index.html` contains the UI shell.
- `style.css` contains the visual design and responsive layout.
- `script.js` contains the game loop, enemies, towers, wave logic, and planet progression.
